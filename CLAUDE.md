# Slic3r Website

## What This Is

A single-page landing site for **Slic3r**, a macOS desktop app that takes 3D models (STL, OBJ, etc.) and slices them into layered cross-sections for laser cutting. The app is built as a Blender plugin (visible in the screenshots — the UI runs inside Blender's interface).

The name is a playful riff on slicing — the mascot is a comic-book superhero called "The Slicer" who "slices everything," generated by Gemini. The whole brand leans into this comic-book energy.

## Design Decisions

### Visual Identity

The color palette is pulled directly from the superhero mascot graphic (`slicer_full.png`):

- **Dark navy/blue backgrounds**: `#0d1f33` (deep), `#1a3a5c` (dark), `#2a5c8f` (mid) — from the character's cape and the moody cityscape
- **Gold/amber accents**: `#d4a825` (gold), `#f0c840` (light gold) — from the character's armor stripes and glowing eyes
- **Panel backgrounds**: `#111827` — for alternating sections

### Typography

- **Bangers** (Google Fonts) for all headings, buttons, and display text — chosen because it's a comic-book lettering font that matches the mascot's speech-bubble style
- **Inter** for body text — clean and readable contrast to the display font

### Comic Book Aesthetic

These subtle details reinforce the theme without being obnoxious:

- Halftone dot pattern overlay on the body (CSS radial-gradient, very faint)
- Diagonal stripe pattern in the hero section (repeating-linear-gradient at -45deg)
- Gold borders and comic-style drop shadows on the hero image (`box-shadow` with hard offset)
- Slight rotation on the hero image that straightens on hover
- A "NEW RELEASE!" burst badge styled like a comic panel callout
- Feature cards with gold border on hover
- Section titles use `text-shadow` for a layered comic-text effect

### Layout

The page has four sections:

1. **Hero** — "THE SLICER! He Slices Everything!" headline with the full mascot graphic. Two CTAs: "Download Now" and "Learn More"
2. **Superpowers** (Features) — Six cards: Multi-Format Import, Precision Slicing, Alignment Markers, SVG & DXF Export, 3D Preview, Blazing Fast
3. **In Action** (Screenshots) — Two real screenshots of the app running in Blender, showing a pig model being sliced into layers. The screenshots have gold borders and captions
4. **Grab Slic3r** (Download) — Single macOS download button (`.dmg package`). Only macOS is supported currently

### Responsive

- On mobile (<768px), the hero stacks vertically (image on top via `column-reverse`), nav links hide, and font sizes scale down

## Deployment Notes

- This is a fully static site — just HTML, CSS, and images. No JavaScript, no build process
- The Google Fonts import (`Bangers` and `Inter`) is the only external dependency — loaded via CSS `@import`
- All image references are relative paths, so the HTML and images just need to be in the same directory
- Download button links are currently `href="#"` — replace with actual `.dmg` download URL
- GitHub link in the footer is also a placeholder `href="#"`
- Should deploy cleanly to Cloudflare Pages with zero configuration (just point it at the directory)

## If You Need to Make Changes

- The gold/blue palette is defined in CSS `:root` variables at the top of the `<style>` block — easy to adjust globally
- Feature descriptions are placeholder copy based on what a slicer app would do — update to match actual features
- The "free and open source" line in the download section should be verified/updated if that's not accurate
- If adding Windows/Linux support later, the download section previously had three platform buttons — just duplicate the macOS button pattern
