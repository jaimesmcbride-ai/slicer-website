A Toolkit for Digital Fabrication

The Toolkit

Each tool targets a different fabrication strategy — pick the one that fits your model

Slicer

Slices a 3D model into a waffle or slab construction — orthogonal interlocking flat plates with half-lap notch joints, ready for laser cutting. Set your slice depth, orientation, and sheet dimensions, then export PDF cut files.

Learn More →
Plate

Detects flat plates in rectilinear models — furniture, boxes, shelving — and generates finger-jointed cut files automatically. The plate detection pipeline runs on load; choose finger, dovetail, or dowel joinery and export.

Learn More →
Paneler

Decomposes a curved 3D surface into flat, developable strips for fabrication from sheet material. Each panel unfolds without distortion. Control the target panel count, max bend angle, and edge joinery style.

Learn More →
Tetris

Fills the interior of a 3D model with a sequence of interlocking tetris-like pieces, producing an animated assembly sequence. Watch each piece drop into place, then export the piece layout for cutting.

Learn More →
Edge

Generates connectors and end-caps for wireframe structures, treating mesh edges as physical boards or rods. Choose from miter, spider, ball, or balloon joinery styles and export the connector geometry for cutting or printing.

Learn More →
Strand

Generates a single continuous chain that folds through a 3D lattice to fill a model — producing an origami-like assembly. Animate the strand folding from flat to finished, then export the layout for assembly.

Learn More →
Crystal

Distributes a repeating node geometry across a crystal lattice fitted to a model's interior or surface. Use the default cube nodes or supply your own STL. Control lattice orientation, scale, and node geometry independently.

Learn More →
One-D

Generates 1D cores — vertical rods or strings — that pass through a 3D model's cross-section for hanging or pin-board assemblies. Choose from string, square, circle, star, and other core shapes. Supports hung, free-standing, and pinboard mount types.

Learn More →
View

A minimal mesh viewer and editor. Load any STL, inspect it for errors with live HUD highlighting, then repair, simplify, relax, or hollow the geometry before passing it to any other Fabricators tool.

Learn More →
