Waffle & Slab Construction for Laser Cutting

Slices a 3D model into orthogonal interlocking flat plates with half-lap notch joints, suitable for laser cutting. Set your slice depth, material thickness, and sheet size — then export PDF or EPS cut files ready for the cutter.

How It Works

From 3D model to laser-ready cut files

Load & Orient

Open any STL model and set the slicing plane orientation — vertical, horizontal, or any angle.

Set Material Params

Specify your stock thickness, sheet dimensions, and scale factor to match your material.

Waffle Mode

Add cross-direction slices for a classic waffle structure with interlocking half-lap joints in both directions.

Variable Spacing

Use the spline editor to vary slice spacing non-uniformly — coarser in flat areas, finer in detailed zones.

Inspect & Animate

Drag the explode slider to spread slices apart, or animate the full assembly sequence.

Export

Output PDF or EPS flat cut-file layouts. Requires Ghostscript for PDF.
