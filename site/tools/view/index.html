Mesh Viewer, Inspector & Editor

A minimal but capable mesh viewer and editor. Load any STL, inspect it with Phong shading and mesh error highlighting, then repair, simplify, relax, or hollow the geometry before passing it to any of the other Fabricators tools.

How It Works

Inspect and prepare meshes before fabrication

Inspect

Loads STL files with Phong shading and a 3-point lighting rig. Mesh errors — non-manifold edges, boundary edges, self-intersections — are highlighted in the HUD legend.

Repair

Edit → Repair fixes non-manifold topology automatically, producing a clean closed mesh ready for any of the fabrication tools.

Simplify

Edit → Simplify decimates the mesh by a specified percentage — useful for reducing high-poly scans to a manageable face count before slicing.

Relax

Edit → Relax smooths geometry while approximately preserving volume, evening out irregular triangulations.

Hollow

Edit → Hollow offsets the surface inward to create a shell, reducing material use for solid models.

Component Selection

⌘ + click selects a connected component. Selected components are highlighted; Backspace deletes them. Export back to STL when done.
